Data & Methodology

Fiscal Year Definition

Fiscal year runs July 1 – June 30. Quarters are defined as:

  • Q1: July – September
  • Q2: October – December
  • Q3: January – March
  • Q4: April – June

Data Sources

  • Attendance & Service Times: Planning Center Online (PCO) via live API. Q3 2025-26 onward is pulled directly from PCO check-in headcounts (Events: Sanctuary 760926, 211 Sumner 760934, 211 YH 760936). Historical data (Q1 2023-24 through Q2 2025-26) from Paul's manually compiled workbook.
  • Baptisms: Manual tracking from Paul's workbook. Includes Infant, Child, and Adult baptisms by worship space.
  • First-Time Guests: From Nicole Burk's ChMS quarterly reports. Tracked by worship space when available.
  • Streaming Hours: Boxcast API via Windsor.ai (minutes viewed converted to hours). Q3 and Q4 2025-26 data is live. Broadcasts classified by worship space using name patterns.

Streaming Metric Note

Beginning Q1 2024-25, we transitioned from tracking "total viewers" and "average viewers" to "hours watched" (total minutes viewed / 60). This metric better reflects engagement depth and allows comparison across different broadcast types (Sunday worship, Ash Wednesday, Lenten midweek, etc.).

Known Data Gaps

  • Sanctuary Q4 2023-24 Attendance: Data not available; shown as "—".
  • FY 2023-24 Baptisms: Sparse data; some quarters incomplete across all spaces.
  • First-Time Guests (FY 2023-24 & early 2024-25): Incomplete tracking; only reported consistently from Q1 2024-25 onward.

Dashboard Notes

  • Charts skip null values; they are not interpolated.
  • Attendance figures represent weekly averages for the quarter.
  • Streaming hours include Sunday worship and special services (Ash Wednesday, Lenten midweek) classified to their worship space.
  • All data is loaded from dashboard-data.js — no external API calls at render time.
